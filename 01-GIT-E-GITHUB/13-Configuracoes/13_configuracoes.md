# Configurações do Repositório

A aba **Settings** é o painel de controle do seu projeto. Aqui você define quem tem acesso, como o código é protegido e integrações externas.

---

## ⚙️ Opções Importantes

### 1. General (Geral)

- **Repository Name:** Mudar o nome do projeto (cuidado, isso quebra links antigos!).
- **Social Preview:** A imagem que aparece quando compartilham seu link no Twitter/LinkedIn.
- **Delete Repository:** A "Zona de Perigo" para apagar o projeto permanentemente.

### 2. Collaborators (Colaboradores)

Aqui você convida pessoas para trabalhar no seu projeto privado ou dá permissão de escrita em projetos públicos.

- **Manage Access:** Adicione usuários pelo nome ou e-mail.

### 3. Branches

- **Default Branch:** Geralmente `main` ou `master`.
- **Branch Protection Rules:** Regras vitais para segurança. Ex: "Ninguém pode fazer push direto na main, precisa de Pull Request" ou "Precisa de 2 aprovações antes de mergear".

### 4. Pages

Configura o **GitHub Pages** para hospedar um site estático diretamente do seu repositório (grátis!).

### 5. Webhooks

Permite que o GitHub avise outros sistemas (como Discord, Slack ou Jenkins) sempre que algo acontecer no repositório.
